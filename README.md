# 基于人工智能与信号处理的动物声音翻译系统

(Animal Sound Translation System Based on AI and DSP)

## 📖 项目简介

本项目是一个融合了**数字信号处理 (DSP)**、**深度学习 (CNN + Transformer 混合架构)** 与 **大语言模型 (LLM)** 的动物声学行为分析与交互系统。

系统不仅能够对 7 类动物（如猴、狗、大象、海豚等）进行高精度的物种识别，还能针对特定物种进行细粒度的**情绪解码**，并最终结合 LLM 生成应对策略与仿生音频反馈，实现“感知-认知-表达”的完整闭环。

------

## 📊 数据集 (Dataset)

本项目的数据集极具多样性，涵盖了陆地与海洋的多种生物声学样本。

- **数据来源：** Kaggle,FreeAnimalSounds,以及部分由小组成员从抖音等小视频平台下载
- **物种构成：** * **全功能组（含情绪标签）：** 猴 (4类情绪)、狗 (3类情绪)、原鸡 (3类情绪)、鹿 (3类情绪)。
  - **单功能组（物种特征展示）：** 大象 (愤怒)、海豚/抹香鲸 (水下声学特征去噪)。
- **数据规格：** 原始音频经过统一清洗，转化为 `22,050 Hz`, `单声道 Mono`, `16-bit PCM` 格式，并统一填充/裁剪为 5.0 秒片段。

## 🚀 快速开始 (Quick Start)

### 1. 环境配置

本项目使用 Python 开发。请确保本地已安装 Python 环境，然后运行以下命令克隆项目并安装依赖：

Bash

```
git clone [https://github.com/](https://github.com/)[你的GitHub用户名]/[你的仓库名].git
cd [你的仓库名]
pip install -r requirements.txt
```

### 2. 准备数据

请先下载上述数据集，并解压至项目根目录的 `dataset_raw/` 文件夹下。随后运行预处理脚本生成特征矩阵：

Bash

```
python src/preprocess.py
python src/feature_extract.py
```

### 3. 运行系统

在项目根目录下执行主程序，启动交互式 GUI 界面：

Bash

```
python src/main_gui.py
```

## 🧠 核心技术栈与创新点

1. **DSP 定制化清洗管道：** 针对不同物种设计专用的**巴特沃斯滤波器**（例如：大象使用 1000Hz 低通，海豚使用 150Hz 高通），并提取 39 维 MFCC 及其动态特征。
2. **Hybrid Fusion 情绪网络：** 创新采用 **CNN (空间特征流) + Transformer (时序特征流)** 的双流混合架构，大幅提升在小样本动物声音数据上的鲁棒性。
3. **闭环式智能交互：** 实现了 `信号输入 -> 情绪解码 -> LLM语义认知 -> 声音生成` 的完整人机交互流。
